Bootcamp 2021: Fun with circles

My first programming project, concluding the 2 week Programming Bootcamp at City (2021).This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, and randomness. I created bouncing circles by creating a loop at the beginning of the code to create multiple of circles. Since I put random in the code where I would control the sizes of the circle,all of their sizes are different and they are able to bounce off the walls.If certain keys are pressed and held on (E,R,T) different colours are produced.For example holding on the key 'T' will make them turn yellow then once you let go off the key the balls will be pink. If the key '1' is pressed the sizes will increase whilst pressing 0 will decrease the sizes of the circle.
